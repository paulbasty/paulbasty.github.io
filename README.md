# ECTIC

A placeholder website for a watch microbrand built from salvaged and swapped parts. Serious about movements, unserious about everything else.

The whole thing is one file, `index.html`, with styles, script, and logo baked in. Open it in any browser and it runs. Nothing to install.

## What is inside

Five pages, switched in the browser with no reloads: Home, Watches, Parts Bin, Story, and List. Each watch has its own detail page with a gallery, spec sheet, pricing, and shipping notes. The watches and casebacks are drawn in CSS as placeholders, ready to swap for real photographs.

## Making it yours

- Products live in the `PRODUCTS` array near the bottom of `index.html`. Edit a name, price, or calibre there and every page updates.
- The logo sits in `LOGO_SRC` as a data URI. Replace it to change the mark in the nav and footer.
- Copy is written to a fixed house style, so keep new text short and plain when you add it.

## Status

Placeholder, iterate freely. Prices, specs, and provenance are invented until the real builds are shot.

See `DEPLOY.md` for putting it online with GitHub Pages.
